Integradora. Introduccion.

El proyecto "Sistema para el registro y consulta de entradas y salidas de personal" tiene como objetivo principal proporcionar a la UTEZ (Universidad Tecnológica Emiliano Zapata) un sistema eficiente y automatizado para el registro y seguimiento de la asistencia del personal. Actualmente, se utilizan lectores de huella digital para registrar las entradas y salidas, pero el personal no tiene acceso a consultar sus propios registros, lo que genera dudas y problemas al momento de aclarar su asistencia.

Placeholder 500x500

Contenido. Integradora.

El sistema propuesto consiste en el desarrollo de una aplicación web que permita activar puntos de registro de entradas y salidas en diferentes lugares físicos de la UTEZ, como CEVICET, Docencia 1, Rectoría y Docencia 4. Además, se implementará un módulo de consulta que permitirá al personal acceder a sus registros personales, y otro módulo para el personal de recursos humanos que les permita consultar los registros de todo el personal y filtrarlos por departamentos, personas y fechas específicas.

Placeholder 500x500

Descripcion. Integradora.

El proyecto implica el desarrollo de una aplicación web funcional que cumpla con los requisitos mencionados anteriormente. Además, se proporcionará una guía de activación y el código fuente completo para facilitar su implementación y mantenimiento. Para la implementación del sistema, se utilizará una base de datos SQL para almacenar los registros de entrada y salida del personal, así como los datos de los empleados. Se proporcionará un script para la creación de la base de datos, asegurando una estructura adecuada y optimizada.
